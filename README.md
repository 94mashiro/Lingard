# ⚽️ Lingard
Lingard 是一款由 Electron 和 React 开发的 Mac 桌面翻译工具。

## 功能
* [x] 调用谷歌翻译接口对原文进行翻译。
* [x] 增加配置界面，可配置语种。
* [ ] 配置本地持久化存储，暂定使用 lowdb。
* [ ] 增加多种翻译接口的支持。
* [ ] 监听剪切板，支持即开即翻。
